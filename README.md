# Prodigy InfoTech Internship - Task 02

## ⏱️ Stopwatch Web Application

This project is part of my Web Development Internship at Prodigy InfoTech.

### 📌 Description
A simple and interactive stopwatch web application built using HTML, CSS, and JavaScript. It allows users to track time, pause, reset, and record lap times.

---

### 🚀 Features
- Start, Pause, and Reset functionality
- Displays time in HH:MM:SS format
- Lap time recording
- Multiple lap tracking
- Clean and responsive UI design

---

### 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

### 🎯 How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Use the buttons to control the stopwatch
